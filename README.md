# Spam Message Detector

This project is a web-based application that detects whether a message is spam or not.

Technologies Used:
- Python
- Flask
- Scikit-learn
- HTML
- CSS

How it Works:
1. User enters a message in the web interface.
2. The message is preprocessed and converted into a vector using a trained vectorizer.
3. A machine learning model predicts whether the message is Spam or Ham.
4. The result is displayed to the user.

Dataset:
SMS Spam Collection Dataset
